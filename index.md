# About PrepFlow

PrepFlow is a modern interview preparation platform designed for developers, students, and tech professionals preparing for technical interviews.

The app provides:
- Curated interview questions
- Interactive quizzes
- Coding practice
- Voice-based interview practice
- Structured learning paths
- Offline learning support

PrepFlow covers multiple technologies and subjects including:
- React Native
- Flutter
- Android
- iOS
- Java
- Kotlin
- Swift
- Dart
- Backend Development
- Computer Science fundamentals

Our goal is to help users improve confidence, strengthen technical knowledge, and prepare effectively for real-world interviews.

PrepFlow is designed for:
